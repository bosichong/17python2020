## config.json 详解

* blog_name : 博客名称
* blog_author : 站长昵称
* blog_description : 站长简介
* meta_description : meta
* meta_keywords : meta keys
* profile_image : 站长头像
* blog_sns : SNS分享
* nav : 博客导航
* blog_list : 首页加载博文篇数


SNS中的ico图标采用的`font-awesome-4.07`，图标搜索地址如下：[http://www.fontawesome.com.cn/faicons/](http://www.fontawesome.com.cn/faicons/),只需填写图标名称即可。
